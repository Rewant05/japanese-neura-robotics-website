# japanese-neura-robotics-website

架空の日本ロボティクス企業「ニューラ・ロボティクス」のために制作した、Next.js、GSAP、WebGL を使った日本語シネマティック Web サイトです。

## 技術構成

- Next.js 16
- React 19
- TypeScript
- GSAP ScrollTrigger
- Three.js / React Three Fiber
- Tailwind CSS

## ローカル実行

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開いて確認できます。

## Vercel デプロイ

このリポジトリを Vercel にインポートすれば、通常の Next.js プロジェクトとしてそのままデプロイできます。
